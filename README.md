# Cognitive Companion

Cognitive Companion is a privacy-first AI system designed to help individuals gain clarity by identifying recurring thoughts and emotional patterns over time.

## What this project does
- Accepts daily journaling inputs (text, voice, screenshots)
- Converts all inputs into text
- Detects basic emotional patterns
- Stores entries locally
- Generates weekly summaries to highlight recurring themes

## What this project does NOT do
- Therapy
- Diagnosis
- Emotional roleplay
- Medical or psychological advice

## Core principle
Insight over advice. Clarity over comfort.

## Status
Phase 1 - Initial development

## Ethics
User data is private by default and is never used to train models.

## Roadmap
- Text journaling
- Emotion detection
- Local memory storage
- Weekly pattern summary
- Voice notes support
- Screenshot (OCR) support
